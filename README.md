# AIonMac 抽签系统 (LotteryApp)

🎉 **AIonMac 抽签系统** 是一款基于原生 SwiftUI 开发的 macOS 桌面端高级抽签应用。专为公司年会、团队建设、大型会议点名等场景设计，提供极致流畅的抽签动画、自适应大屏的动态阵型排版以及极度舒爽的键盘交互体验。

## ✨ 核心特性

- **🚀 智能金字塔阵型排版 & 大屏自适应**
  彻底告别杂乱无章的布局！内置智能排版算法，无论单次抽取 1 人还是满屏 20 人，系统都会自动计算最优的行列组合（如 3人呈“1上2下”，10人呈“3-3-4”，20人呈“5-5-5-5”）。
  配合动态几何缩放 (`GeometryReader`)，**完美适配 2K、4K 及投影大屏**，卡片和字体会自动等比放大，绝不拉伸变形。
- **⌨️ 极客级键盘交互（全局防丢失）**
  无需鼠标瞄准，按下 **空格键 (Space)** 即可一键开始/停止抽签！甚至在你点击了副屏幕（切走了焦点）的情况下，依然可以通过空格键从容停止抽签。
- **👁️ 强迫症级别的视觉优化**
  - **姓名防抖跳跃：** 抽签滚动时，系统会自动为“两个字的姓名”中间插入全角空格对齐“三个字的名字”，彻底消除滚动时标签闪烁跳跃的视觉痛点。
  - **按钮防褪色：** 彻底重写了系统级按钮渲染，即使窗口失去焦点，停止按钮依然保持鲜艳的红色，随时待命。
  - **启动即全屏：** 软件打开后自动沉浸式全屏，直接进入展示状态。
- **📊 CSV 数据轻松导入**
  支持一键导入 UTF-8 编码的 `.csv` 表格，自动进行去重、去空值校验。
- **🎯 灵活的抽签规则**
  支持按分类抽取（系统自动识别筛选 AE 或 SE），支持“允许重复中签”与“不可重复中签”模式。
- **📝 完善的历史记录管理**
  独立且无遮挡的侧边栏历史记录，自动按时间倒序排列，并支持一键导出为 CSV 格式用于留档和发放奖品。

## 📋 数据导入格式要求

请使用 Excel 或 Numbers 创建表格，并**另存为 CSV (UTF-8) 格式**。
表格**第一行必须包含以下完整的四个表头**（顺序不限）：

| 组名 | 公司名 | 姓名 | 用户类型 |
| :--- | :--- | :--- | :--- |
| 第一组 | 苹果公司 | 张三 | AE |
| 核心组 | 微软公司 | 欧阳修 | SE |
| 业务组 | 谷歌公司 | 李　四 | AE |

> **💡 提示：** 
> 1. 表头名称必须完全匹配 `组名`、`公司名`、`姓名`、`用户类型`。
> 2. `用户类型` 仅支持填写 **AE** 或 **SE**。**无需在表格中填写“全部”，系统会自动为你生成“全部”选项，以便一键同时抽取 AE 和 SE 人员。**

## 💻 运行环境与技术栈

- **平台**: macOS 13.0+ (原生 Mac 桌面应用)
- **界面框架**: SwiftUI
- **并发处理**: Swift Concurrency (`async/await`, `Task`)
- **数据流**: Combine & `@StateObject`
- **底层支持**: 深度调用 `NSEvent` 全局监听与 `NSWindow` 渲染层管理
